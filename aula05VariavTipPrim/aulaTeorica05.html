Trabalhando as váriáveis com JavaScript

Comentários:

  • // Usado para uma linha
  • /**/ Usado para mais de uma linha (Usado para depuraçao também)

Variáveis:

  • Vaga a1 = carro1
  • a1 = null
  • Pode ser usado três tipos de aspas (", ', `)
  • O nome de cada variável é chamado de IDENTIFICADORES REGRAS:
    • Pode começar com letras; $ ou _
    • NÃO pode começar com números
    • É possivel usar letras e números
    • É possível usar acentos e símbolos
    • NÃO pode usar palavras reservadas (JavaScript usa como comando)

Tipos Primitivos:

    Tpos Primitivos:
  • 5, 8, -12, 0.5 -15.9, 3.14 (Para o JavaScript são considerados como 'NUMBERS'), não há distinção de inteiro, real ou float.
  • "Google", 'JavaScript', `Maria`(São as STRING), usando os três tipos de aspas.
  • true, false (São os BOOLEAN)
  • Os NAMBERS se dividem em:
    • Infinity
    • NaN
  • null (Considerado como um objeto)
  • undefined
  • object:
    • Array
  • function
  • typeof (Tipo de)
