Conversão de STRING para NÚMEROS:

  • Number.parseInt(n), converte uma string em números
  • Number.parseFloat(n), converte em números reais com virgulas(flutuantes).
  • number(n), nesse caso o JavaScript decide qual é o número.

Conversão de NÚMEROS para STRINGS:

  • Strin(n)
  • n.toString

Formatando as casas decimais dos números:

  • 'toFixed(2), ajusta as casas decimais
  • 'toFixed(2).replace('.', ','), troca o ponto por virgulas
  • toLocaleString(pt-BR, {style: 'currency', currency: 'BRL'}), PARA COLOCAR O 'R$' antes do número.
  • Resultado node.js: > valor = 1550.2 1550.2 > valor.toFixed(2) '1550.20' > valor.toFixed(2).replace('.', ',') '1550,20' > valor.toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'}) 'R$ 1.550,20' >
